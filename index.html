This website uses simple JavaScript to check your system time. Any website could do this and this can be used to fingerprint you by your clock offset and clock skew. This can also be used to figure out your location by determining which countries have similar time as you.

Refresh the page to update the time.

If you set your system time back, you can see that this picks it up too. This does not determine the time from the system timezone.
